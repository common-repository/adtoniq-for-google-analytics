== Description ==

Adtoniq For Google Analytics allows WordPress sites to make full use of Google Analytics even with their blocked audience, and also allows customers to segment their audience into blocked and non-blocked users.

Highlights:

= Get the full picture =
Ensures that all the same Google Analytics data is collected for both non-blocked and blocked users alike.

= Uses your Google Analytics account =
Works with your own Google Analytics account. If you don't have one, we can help you get set up.

= Uses custom dimensions for reporting =
Uses custom dimensions for reporting, so you can understand differences in behaviour and demographics between your blocked and unblocked web.

= Supports traffic splitting =
Supports traffic splitting so you can stay within the Google Analytics quota.

== Installation ==

This plugin requires that you first install and activate the [Adtoniq plugin](https://wordpress.org/plugins/adtoniq/) plugin. After installing that,
install and activate this plugin. Click to the Adtoniq for Google Analytics admin page and enter your Google Analytics property ID, traffic split,
and check the box if you want Google Analytics to work even with ad blockers.
